1. Basic OSM setup

Minimal controller with default config. Fist pans, two open hands zoom.

Live preview Gestures off
TypeScript
import { GestureMapController } from '@map-gesture-controls/ol';

const controller = new GestureMapController({ map });

// must be called from a user gesture (button click)
await controller.start();

controller.stop(); // tear down webcam and overlay

2. Toggle button with disabled state

Single button starts and stops gesture control. Disabled while loading to prevent double-clicks.

Live preview Gestures off
TypeScript
const controller = new GestureMapController({ map });
const btn = document.getElementById('toggle-btn') as HTMLButtonElement;
let active = false;

btn.addEventListener('click', async () => {
  btn.disabled = true;
  if (active) {
    controller.stop();
    active = false;
    btn.textContent = 'Start gesture control';
  } else {
    await controller.start();
    active = true;
    btn.textContent = 'Stop gesture control';
  }
  btn.disabled = false;
});

3. Custom webcam overlay position

Webcam overlay moved to bottom-left, smaller size, reduced opacity. Enable gestures to see the difference.

Live preview · webcam: bottom-left 200×150 opacity 0.6 Gestures off
TypeScript
const controller = new GestureMapController({
  map,
  webcam: {
    position: 'bottom-left',
    width:    200,
    height:   150,
    opacity:  0.6,
  },
});

4. Adjusted gesture sensitivity

Lower dwell time for faster gesture confirmation, tighter dead zones for more responsive pan and zoom.

Live preview · tuning: faster dwell, tighter dead zones Gestures off
TypeScript
const controller = new GestureMapController({
  map,
  tuning: {
    actionDwellMs:     50,    // confirm gestures faster (default 80 ms)
    releaseGraceMs:    100,   // shorter grace period (default 150 ms)
    panDeadzonePx:     5,     // smaller hand movement threshold (default 10)
    zoomDeadzoneRatio: 0.003, // finer zoom control (default 0.005)
  },
});
